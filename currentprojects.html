PantryPal

My first solo project I worked on. This application lets a user input their ingredients and it pulls up a recipe that the user can make based on the ingredients that they have. I plan on polishing this application further, adding more recipes or possibly an API. I also would like to add more user functions such as users adding their own recipes, a liked recipe system, and a shopping list system that could link to Instacart.
